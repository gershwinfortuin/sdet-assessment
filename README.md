# sdet-assessment

This repository is used for personal use as well as completing assessments.

##Installation

- run 'npm install'
- Isntall codecept.js
- run 'npx create-codeceptjs .'
- run 'npx codeceptjs init'
- Select puppeteer

## Running the test

- npm run test_run

- run 'allure serve output' to display the report
